# Advising Appointment System

This repository contains the Advising & Appointment System built with React Native and PostgreSQL.

## Features
- Role-based authentication (student, advisor)
- Student and advisor profiles
- Advisor availability calendar and slot generation
- Appointment booking and management
- Walk‑in queue system
- Case notes and referral system
- Real‑time notifications

## Project Structure
```
backend/        # Server (Node.js/Express) with PostgreSQL
frontend/       # React Native app
```

## Setup
1. Clone the repo
2. Install dependencies
3. Configure environment variables
4. Run migrations and start the server
5. Run the React Native app
